Hydrogeological Toolbox

Data preparation

Well test often requires to treat the data prior to their interpretation. One of the strength of hytool is to make full benefit of the matlab environment to preprocess the data. Functions that are specially helpful are the import capabilities (load, xlsread, cvsread), interpolation functions (interp1), ploting functions (plot) and the signal processing tools.

In addition, in order to facilitate some operations that are very frequent, hytool provides a set of simple utilies that are listed below.

ldf Load Data File: reads a two column ASCII file
igmeread Read the time, head, and flow rate from the IGME standard csv file
hyclean Clean the data sets by selecting values that are finite and have strictly positive time
hyselect Selecting a part of a dataset between xstart and xend
hyfilter Filtering a signal in order to reduce the noise
hysampling Sampling a signal at regular intervals
hyplot Plotting selected/filtered/sampled data
simplify_rate Group time periods having the same flow rate
